一つ前のクエストでエディタのターミナルでgitコマンドを学習したおかげで、
今までsourcetreeを使っていたけど、ようやくターミナルの楽さに気づきました。

 ** よく使うコマンド ** 

現在のブランチの確認： `git branch`
ブランチの切り替え： `git switch ブランチ名(mainなど)`
最新の状態にpull： `git pull`
ブランチを作成して切り替え： `git switch -c ブランチ名(mainなど)`
変更をコミット： `git add . `
コミットのメモ： `git commit -m "コミットにつけるメモ"`
変更をpush： `git push origin ブランチ名(mainなど)`
