# TIL (Today I Learned)

日々の学びを記録していくリポジトリです。

## ルール

- 学んだことを簡潔にまとめる
- 技術的な学びは、カテゴリごとにフォルダ分け
- その他の学び・気づきは日報的な記録で OK！
- 完璧を求めず、小さな記録を習慣化


## 自分の学習フローについて

- インプット（学ぶ）
  簡単なアプリを作りながら、知らない記述をAIに聞いたり検索したりして基礎を学ぶ。
- メモ・整理（理解する）
  TILをちゃんと活用して、学習した内容を書き留める。
- アウトプット（発信する）
  学習したメモをコミット・プッシュ。

### 心がけ
- 働きながらの学習なので、無理をしない。
- 学習を楽しむ。無理はしないけど自分の身につくことを心がける。
- サボるモードにならないように、あまり間隔を空けない。
