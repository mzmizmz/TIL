# Git ブランチについて

## ブランチとは

Git のブランチは、作業の流れを分岐させる機能。

複数の作業をお互いに影響を与えずに、並行して進めることができる。

## 基本コマンド

- `git branch`: ブランチ一覧を表示
- `git branch <ブランチ名>`: 新しいブランチを作成
- `git switch <ブランチ名>`: ブランチを切り替え
- `git switch -c <ブランチ名>`: ブランチを作成して切り替え
- `git merge <ブランチ名>`: ブランチをマージ

## なぜ重要か

- チーム開発で必須のスキル
- main ブランチを安全に保てる
- 作業を整理しやすい
